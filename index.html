Abstract
<<<<<<< HEAD Activity recognition fits within the bigger framework of context awareness. In this project, we report on our efforts to recognize user activity from accelerometer data. The human physical activities which we have considered for our classification are standing, sitting, walking, cycling, stairs climbing and running. Firstly, we have done feature extraction from our data and this is further interpreted by various machine learning Algorithms. The basic machine learning task involves classifying a tri-axial time series as a single physical activity type. ======= This project describes how to recognize certain types of human physical activities such as walking, stairs ascending/descending using 3 axis acceleration data generated by a 3 axis accelerometer being put at waist. The system was trained and tested in an experiment with multiple human subjects in real-world conditions. Several classifiers were tested using various statistical features. High-frequency and low-frequency components of the data were taken into account. We selected classifiers each offering good performance for recognizing our set of activities and investigated how to combine them into an optimal set of classifiers. >>>>>>> 1afea87eefa8fd14710cb7e52deabb72ebfb9fb4
